## Description
basic QPSK modulated channel model for learning digital communication
## Pre-requisites
- MATLAB 2018

## Dependencies
- pnrz.m
- QPSK_mod.m
- QPSK_demod.m

## Installation and Run
Make sure to install Pre-requisites and availability of dependencies. <br/>
Run topmodel.m file

## Functions
- bit generation
- pnrz line coding
- QPSK Modulation
- Matched filter demodulation
